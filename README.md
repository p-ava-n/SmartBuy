SMARTBUY

PROJECT TITLE

Simple e-commerce web application

ABOUT

SmartBuy is an e-commerce website built using HTML, CSS, and JavaScript.
It allows users to browse products, add items to a cart, and simulate a checkout process.

This project was created as part of learning frontend development.

FEATURES IMPLEMENTED

1. Product listing with categories
   
2. Search and filter functionality
   
3. Add to cart
   
4. Increase / decrease item quantity
   
5. Remove items from cart
   
6. Dynamic total price calculation
   
7. Promo code support

8. Checkout form (Shipping + Payment + Review)

TECHNOLOGIES USED

HTML

JavaScript

CSS

RUNNING THE PROJECT

1. Download or clone the project folder


2. Make sure all files are in the same directory:

   
   index.html


   style.css


   script.js


3. Open index.html in your browser
